# Feedback Learn Playbook

## Purpose
Capture durable lessons from review outcomes.

## Command
`catalog learn --run <job-id>`

or

`catalog learn --lesson "<lesson>"`

## Inspect
- `.catalog/learning/catalog-learning.md`

## Agent behavior
- Add only stable, reusable lessons.
- Avoid dumping raw event logs into the learning file.
- Keep learning entries short and actionable.
